# Reading Journal

Este é um projeto de **Diário de Leitura** desenvolvido com **React**, que permite cadastrar, listar e remover livros lidos.

## Funcionalidades

- **Listagem de Livros**  
- **Cadastro de Livros**  
- **Remoção de Livros**
